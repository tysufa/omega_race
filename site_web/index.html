A l'occasion du cours de concrétisation de la licence d'informatique de l'université d'Angers, chaque étudiant fut répartit en binôme. Chaque binôme se vit assigné un projet de programation dans le but d'apliquer les notions apprisent au cours de l'année. En plus de ce projet, les binômes devaient élaborer un site web affin de présenter leur projet. C'est ainsi que nous avons l'honneur de vous acceuilir sur la page officielle du projet

est un jeu d'arcade de type "shoot'em up" conçu par Ron Haliburton et publié en 1981 par Midway. Le but du projet est de recreer le jeu .
Dans le jeu original, le joueur contrôle un vaisseau spacial ayant pour but de détruire les vaisseaux adverses au sein d'une arène rectangulaire. Le joueur disposes de boutons pour orienter son vaisseau, un pour activer les propulseurs, et un pour tirer. Le vaisseau rebondit sur un mur invisible aux bords de l'écran qui apparait brievement lorqu'il est touché.
Les consignes du projet demandaient de conserver les aspects fondamentaux du jeu :

  • Le thème
  • Le vaisseau du joueur
  • Les rebonds sur les murs et le cadre central
  • Les ennemis peuvent se chevaucher
  • Le score et les vies
